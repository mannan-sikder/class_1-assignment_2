HTML elements reference


This page lists all the HTML elements, which are created using tags. They are grouped by function to help you find what you have in mind easily. An alphabetical list of all elements is provided in the sidebar on every element's page as well as this one.

Content sectioning


Content sectioning elements allow you to organize the document content into logical pieces. Use the sectioning elements to create a broad outline for your page content, including header and footer navigation, and heading elements to identify sections of content.


Content sectioning
Serial Tag name <> Description Example
1 <address> The HTML <address> element indicates that the enclosed HTML provides contact information for a person or people, or for an organization. <address>
sikder021@gmail.com
+8801922421768 </address>
2 <article> The HTML <article> element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable <article>This is a article </article>
3 <aside> The HTML <aside> element represents a portion of a document whose content is only indirectly related to the document's main content. <aside >

The Rough-skinned Newt defends itself with a deadly neurotoxin.

</aside >
4 <header> The HTML <header> element represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements. <header>This is header section </header>
5 <h1>,<h2>,<h3>... <h6> The HTML <h1>–<h6> elements represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest. <h1>This is the header </h1>
<h2>This is the header </h2>
..................
<h6>This is the header </h6>
6 <hgroup> The HTML <hgroup> element represents a multi-level heading for a section of a document. It groups a set of <h1>–<h6> elements. <hgroup>
<h1>Calculus I</h1>
<h2>Fundamentals</h2>
</hgroup>
7 <main> The HTML <main> element represents the dominant content of the body of a document. The main content area consists of content that is directly related to or expands upon the central topic of a document, or the central functionality of an application. <main>This is main section </main>
8 <nav> The HTML <nav> element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes. <nav>This is nav section </nav>
9 <section> The HTML <section> element represents a standalone section — which doesn't have a more specific semantic element to represent it — contained within an HTML document.
10 <footer> The HTML <section> element represents a footer for its nearest sectioning content or sectioning root element. A footer typically contains information about the author of the section, copyright data or links to related documents. <footer>This is footer section </footer>
